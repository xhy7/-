# Developer 1 协作说明

## 负责范围
- 独占维护 `src/app`、`src/shared`、`src/mocks`。
- 负责项目脚手架、页面组装、共享 UI、类型合同、mock gateway、测试集成。

## 必须完成的事情
- 固定首页装配顺序和页面骨架。
- 维护全局样式变量和共享原子组件。
- 保证 `src/shared/contracts` 是其他人唯一依赖的数据接口。
- 提供稳定 mock 数据，不让功能区开发者自行造协议。

## 不要做的事情
- 不把功能区逻辑写回 `src/app/page.tsx`。
- 不绕开合同直接给功能区塞临时 props。
- 不在未通知的情况下改动功能区目录内代码。
